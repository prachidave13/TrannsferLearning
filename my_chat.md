This is a networking/subnetting question. Here's how to calculate the subnet address:

Steps to Find Subnet Address:
Given:

IP Address: 150.215.17.9
Subnet Mask: 255.255.240.0
Method: Perform a bitwise AND operation between the IP and Subnet Mask

Convert to Binary:
IP Address: 150.215.17.9

150 = 10010110
215 = 11010111
17  = 00010001
9   = 00001001
Subnet Mask: 255.255.240.0

255 = 11111111
255 = 11111111
240 = 11110000
0   = 00000000
Perform AND Operation:
IP:     10010110.11010111.00010001.00001001
Mask:   11111111.11111111.11110000.00000000
        ─────────────────────────────────────
Result: 10010110.11010111.00010000.00000000
Convert Back to Decimal:
10010110 = 150
11010111 = 215
00010000 = 16
00000000 = 0
Subnet Address = 150.215.16.0 ✓

Quick Method:
For the third octet:

17 in binary: 00010001
240 in binary: 11110000
AND result: 00010000 = 16
So you keep 150.215, replace 17 with 16, and set the last octet to 0.

Answer: 150.215.16.0
